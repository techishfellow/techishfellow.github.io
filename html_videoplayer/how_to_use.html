How To Use

Player HowToUse FollowMe DownloadScript
  • Download 'HTML Video Player Script' from GitHub or Our Official Website
  • Then extract the zip file using any zipextractor software

  • Open the newly extracted folder
  • Rename your video file to 'video.mp4', 'video.webm' or 'video.ogg'
  • Then paste your video file in 'Paste Your Video Here' folder
  • Now Open the 'index.html' file via any browser and tap on the play button to enjoy your video :-)

  • Paste only one file in the folder at a time (Muiltiple file support not available)
  • Supported Video Formats:   mp4, webM, ogg
  • Supported Video Resolutions:   144p to 4320p (30-60 FPS)
